Where's my key

An FPS game where the player is time bound to escape the room by finding the right key

SCROLL DOWN

Where's my Key GDD

Software Used
  • Unity Game Engine
  • Blender
  • Adobe photoshop

Genre

First Person, Survival, Single Player, Puzzle, Exploration

Platform

Windows/PC

Description

Our player is stuck inside a room and bomb is about to explode. The Player have to find the real key among the fake one by solving the puzzles present in the room. The Player needs to escape the room before the timer counter reaches ‘0’. Puzzles are written in the notes which can be picked up by the player.



Game Features

Timer Counter

The Player has been given 06 minutes to escape the room and win. Once the timer reaches ‘0’, “Game Over” screen will be displayed.

Notes

The Player is required to pick up various notes and solve puzzles in order to find the real key in the room.

  • Only 1 is the Real key that’ll unlock the door
  • Rest keys are fake and won’t open the door.

Atmosphere

The atmosphere is surrounded by fog and particles. Thus, making it difficult for the player to locate the keys.

Interaction

There are multiple objects where the player can move/interact like:

  • Drawers can be opened/closed
  • Books can be moved
  • Cushions can be moved

Rest some items will only be moveable after solving puzzles

*There is also one hidden easter egg in the game*

Winning Condition

The player has to escape the room with the right key in order to win the game before the time runs out.

Player Characteristics

  • The player spawns in a room.
  • The player needs to find the correct key by solving the puzzles.
  • The game ends once the player unlocks the door and escapes.
  • The game can be restarted after losing or time runs out.
  • The player can interact/move different objects in the house.
  • The player can run and jump to spot the house.
  • The player can run pick up notes and solve puzzles.
  • Keys can be found everywhere in the house but only '1' of them works.

Core Loop

Title Screen > Instructions > Player enters the gameplay > Solve the puzzles > find the key > escape room > End Screen


Goals

Find the real key among the fake keys by solving the puzzles in order to escape the room before the time runs out.
